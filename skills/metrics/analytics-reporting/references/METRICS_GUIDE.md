# DevRel Metrics Guide
## AAARRRR Framework
- Awareness: How do devs discover us?
- Acquisition: How do they sign up?
- Activation: First value moment
- Retention: Do they come back?
- Revenue: Do they pay?
- Referral: Do they recommend?

## Key Metrics
- Time to First Hello World
- API call growth
- Community engagement
- NPS score
