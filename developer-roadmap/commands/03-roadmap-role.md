# /roadmap-role - Explore Specific Roles

Get detailed information about a specific developer role. Learn skills, resources, and career paths.

## Usage

Specify your role of interest:

```
/roadmap-role frontend
/roadmap-role backend
/roadmap-role devops
/roadmap-role data-science
/roadmap-role machine-learning
/roadmap-role engineering-manager
/roadmap-role blockchain
/roadmap-role game-developer
```

## Example: Backend Developer

### Skill Stack
- **Languages**: Node.js, Python, Java, Go
- **Databases**: PostgreSQL, MongoDB, Redis
- **APIs**: REST, GraphQL, gRPC
- **Testing**: Unit tests, integration tests
- **DevOps**: Docker, basic CI/CD
- **Soft Skills**: Communication, problem-solving

### Learning Path
1. Choose primary language
2. Master basics (3 months)
3. Build API projects (3 months)
4. Add database skills (2 months)
5. Learn testing & DevOps (2 months)
6. Build portfolio projects (ongoing)

### Jobs Available
- Junior Backend Developer
- Backend Engineer
- Full Stack Developer
- API Developer
- Microservices Developer

### Key Concepts
- HTTP & REST principles
- Database design
- API security
- Scalability
- Testing strategies

## Role Selection Criteria

### Choose Based On:

1. **Interest Level**
   - What excites you?
   - What problems engage you?
   - What's your learning style?

2. **Skills Match**
   - Current programming experience
   - Mathematical ability
   - Problem-solving strength
   - Domain knowledge

3. **Market Factors**
   - Job availability
   - Salary expectations
   - Company demands
   - Growth trajectory

4. **Career Goals**
   - IC vs management
   - Specialization vs generalization
   - Domain focus
   - Impact desired

## Progression Paths

**Individual Contributor Path:**
Role → Senior Role → Staff/Principal → Executive

**Leadership Path:**
Role → Tech Lead → Engineering Manager → Director → VP

**Specialist Path:**
Role → Deep Specialization → Expert → Thought Leader
