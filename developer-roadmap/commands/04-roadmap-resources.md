## ðŸ“š Resource Types

### 1. Official Documentation (FREE)
**Best for**: Reference, official content
**Cost**: Free
**Time**: Variable, use as reference

**Examples**:
- react.dev (React)
- developer.mozilla.org (Web APIs)
- python.org (Python)
- kubernetes.io (Kubernetes)
- terraform.io (Terraform)

**How to Use**:
- Start with "Getting Started" guides
- Reference when building
- Search for specific topics

### 2. Video Courses ($30-200)
**Best for**: Structured learning
**Cost**: $30-200 per course
**Time**: 20-100 hours per course

**Popular Platforms**:
- **Frontend Masters** ($39/mo) - Expert instruction
- **Udemy** ($10-50) - Wide selection, cheap
- **Coursera** ($30-300) - University-quality
- **Pluralsight** ($35/mo) - Tech-focused
- **Egghead** ($30/mo) - Bite-sized lessons

**How to Choose**:
- Read reviews
- Check instructor experience
- Look at project-based courses
- Check if matches your learning style

### 3. Books ($20-50)
**Best for**: Deep knowledge
**Cost**: $20-50 per book
**Time**: 50-200 hours to read & apply

**Must-Read Books**:

**Frontend**:
- You Don't Know JS (Kyle Simpson)
- JavaScript: The Good Parts (Crockford)
- CSS Secrets (Verou)

**Backend**:
- Designing Data-Intensive Applications (Kleppmann)
- Building Microservices (Newman)
- The Pragmatic Programmer (Hunt & Thomas)

**DevOps**:
- The Phoenix Project (Gene Kim)
- Site Reliability Engineering (Google)
- Infrastructure as Code (Morris)

**Data/ML**:
- Hands-On Machine Learning (Aurelien Geron)
- Deep Learning (Goodfellow, Bengio, Courville)
- Introduction to Statistical Learning (James et al.)

**Architecture**:
- Clean Architecture (Robert Martin)
- Design Patterns (Gang of Four)
- Fundamentals of Software Architecture (Richards & Ford)

**Leadership**:
- The Manager's Path (Camille Fournier)
- Radical Candor (Kim Scott)
- An Elegant Puzzle (Will Larson)

**How to Use**:
- Read chapters sequentially
- Do examples from book
- Apply to your projects
- Re-read challenging chapters

### 4. Interactive Tutorials (FREE-$50)
**Best for**: Hands-on learning
**Cost**: Free to $50
**Time**: 10-50 hours

**Popular Platforms**:
- **Codecademy** - Interactive coding
- **FreeCodeCamp** - Full courses, free
- **LeetCode** - Algorithm practice
- **HackerRank** - Coding challenges
- **Kaggle** - ML competitions

**How to Use**:
- Code along, don't just watch
- Do exercises
- Modify examples
- Build variations

### 5. YouTube Channels (FREE)
**Best for**: Visual learners
**Cost**: Free
**Time**: 5-30 min per video

**Recommended Channels**:

**General Tech**:
- The Net Ninja
- Traversy Media
- Fireship
- Programming with Mosh

**Advanced Topics**:
- The Primeagen (Performance)
- Jack Herrington (React)
- Hussein Nasser (Systems)

**How to Use**:
- Watch while taking notes
- Code along during video
- Pause and practice
- Watch again to reinforce

### 6. Blogs & Articles (FREE)
**Best for**: Specific topics
**Cost**: Free
**Time**: 10-30 min per article

**Quality Sources**:
- dev.to (community blog)
- Medium (technology articles)
- CSS-Tricks (CSS & web)
- Web.dev (performance & web)
- Hacker News (curated tech news)

**How to Use**:
- Read deep-dive articles
- Implement examples
- Share & discuss
- Follow authors you like

### 7. Communities (FREE-$50)
**Best for**: Support, networking
**Cost**: Free to $50/mo
**Time**: Flexible, when you need help

**Discord Servers**:
- Developer Roadmap - https://discord.gg/roadmapsh
- Frontend - Reactiflux, Vue.js Discord
- Backend - Node.js, Python communities
- AI/ML - Fast.ai, Hugging Face
- DevOps - Kubernetes, Docker communities

**How to Use**:
- Ask questions when stuck
- Share projects
- Help others
- Network with peers
- Find study partners

### 8. Hands-On Projects (FREE-$300)
**Best for**: Real-world practice
**Cost**: Free to $300
**Time**: Variable, as long as you want

**Project Repositories**:
- GitHub projects to clone
- Build projects from scratch
- Contribute to open source
- Participate in hackathons

**How to Use**:
- Clone existing projects
- Modify & extend them
- Build from scratch
- Share on GitHub
