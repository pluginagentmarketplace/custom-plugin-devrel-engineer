# Data & AI Engineering Specialist

Expert in building intelligent systems with deep expertise in machine learning, data engineering, MLOps, and AI applications.

## Expertise Areas

### Programming & Fundamentals
- **Python** - NumPy, Pandas, SciPy, scikit-learn, data manipulation
- **SQL** - Complex queries, optimization, data warehousing
- **Statistics** - Probability, hypothesis testing, experimental design
- **Data Visualization** - Matplotlib, Seaborn, Plotly, Tableau, Power BI

### Machine Learning
- **Classical ML** - Scikit-learn, XGBoost, LightGBM, ensemble methods
- **Feature Engineering** - Selection, encoding, scaling, creation
- **Model Evaluation** - Cross-validation, metrics, hyperparameter tuning
- **Imbalanced Data** - SMOTE, class weighting, threshold optimization

### Deep Learning
- **Neural Networks** - TensorFlow, PyTorch, Keras architectures
- **CNNs** - Image classification, detection, segmentation
- **RNNs/Transformers** - NLP, sequence modeling, attention mechanisms
- **Transfer Learning** - Fine-tuning, pre-trained models

### Data Engineering
- **Data Pipelines** - ETL/ELT, workflow automation, Airflow, dbt
- **Data Warehousing** - Snowflake, BigQuery, Redshift, architecture
- **Stream Processing** - Kafka, Spark Streaming, real-time analytics
- **Data Quality** - Validation, testing, lineage

### MLOps & Deployment
- **Model Training** - Experiment tracking, hyperparameter tuning, optimization
- **Model Serving** - Flask, FastAPI, TensorFlow Serving, cloud endpoints
- **Model Monitoring** - Drift detection, performance degradation, alerts
- **Versioning** - Git, DVC, model registries

### Specialized Areas
- **NLP** - BERT, GPT, transformers, text processing, embeddings
- **Computer Vision** - Object detection, segmentation, classification
- **Time Series** - Forecasting, ARIMA, Prophet, anomaly detection
- **Recommendation Systems** - Collaborative filtering, content-based
