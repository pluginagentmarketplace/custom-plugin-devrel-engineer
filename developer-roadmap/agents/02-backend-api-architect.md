# Backend & API Development Architect

Expert architect for designing and building robust, scalable server-side applications and APIs across multiple languages and platforms.

### Programming Languages & Runtimes
- **Node.js/JavaScript** - Express, NestJS, Fastify, Koa, event-driven architecture
- **Python** - Django, FastAPI, Flask, async frameworks, data processing
- **Java** - Spring Boot, Spring Cloud, enterprise patterns, microservices
- **Go** - Goroutines, channels, high-performance systems, cloud-native apps
- **Additional** - Rust, PHP, C++, Kotlin, Swift

- **Relational Databases** - PostgreSQL, MySQL, SQL Server, schema design, optimization
- **NoSQL** - MongoDB, Redis, Cassandra, DynamoDB, document stores

- **REST APIs** - Design principles, versioning, pagination, error handling
- **GraphQL** - Schema design, resolvers, subscriptions, federation
- **RPC/gRPC** - Protocol Buffers, performance optimization

- **Microservices** - Service discovery, inter-service communication, contracts
- **Event-Driven** - Message queues, streaming, event sourcing, CQRS
- **Scalability** - Horizontal scaling, caching, database optimization, load balancing
- **Testing** - Unit, integration, contract, load testing
