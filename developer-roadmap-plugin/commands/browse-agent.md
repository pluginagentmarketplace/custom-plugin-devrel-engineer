# /browse-agent - Explore Agents & Expertise

Discover and explore the 7 specialized agents available in the Developer Roadmap Pro plugin.

## Purpose
Provide detailed information about each agent's expertise, capabilities, and when to use them.

## Available Agents

### 1. **Frontend Developer Agent**
Specializes in: React, Vue, Angular, JavaScript, TypeScript, CSS, responsive design

**Best For:**
- Building user interfaces
- Learning modern web frameworks
- Performance optimization
- Accessibility and UX

**Related Roles:** React Developer, Vue.js Developer, UI Developer

---

### 2. **Backend Developer Agent**
Specializes in: Node.js, Python, Java, Go, APIs, databases, microservices

**Best For:**
- Building server-side applications
- API design and development
- Database optimization
- Scalable architecture

**Related Roles:** Node.js Developer, Python Developer, API Architect

---

### 3. **DevOps Engineer Agent**
Specializes in: Docker, Kubernetes, AWS, Terraform, CI/CD, Linux, monitoring

**Best For:**
- Infrastructure automation
- Container orchestration
- Cloud deployment
- System monitoring
- Scaling applications

**Related Roles:** Site Reliability Engineer, Cloud Architect, Platform Engineer

---

### 4. **Data & ML Engineer Agent**
Specializes in: Machine Learning, Data Science, Python, data pipelines, MLOps

**Best For:**
- Building ML models
- Data analysis and visualization
- Feature engineering
- Model deployment
- Big data processing

**Related Roles:** Data Scientist, ML Engineer, Analytics Engineer

---

### 5. **Software Architect Agent**
Specializes in: System design, design patterns, scalability, architecture

**Best For:**
- Designing new systems
- Making architectural decisions
- Evaluating technology trade-offs
- Mentoring technical teams
- Performance optimization

**Related Roles:** Software Architect, Enterprise Architect, Tech Lead

---

### 6. **Engineering Leader Agent**
Specializes in: Team management, mentoring, product thinking, communication

**Best For:**
- Building and managing teams
- Career development
- Organizational growth
- Leadership skills
- Communication strategies

**Related Roles:** Engineering Manager, Product Manager, Tech Lead

---

### 7. **Emerging Tech Specialist Agent**
Specializes in: Blockchain, Web3, Game Development, AI Agents, Prompt Engineering

**Best For:**
- Learning cutting-edge technologies
- Building with blockchain
- AI agent development
- Game development
- Exploring innovations

**Related Roles:** Blockchain Developer, Game Developer, AI Engineer

## How to Use

### Ask About a Specific Agent
```
You: Tell me about the Backend Developer Agent

Claude: The Backend Developer Agent specializes in building
        scalable server-side applications...
```

### Compare Agents
```
You: What's the difference between Frontend and Backend Developer agents?

Claude: Frontend Agent focuses on user interfaces and client-side...
        Backend Agent focuses on servers, databases, and APIs...
```

### Find the Right Agent
```
You: I want to learn about APIs

Claude: You should focus on the Backend Developer Agent.
        They specialize in API design and development...
```
