# /learn - Interactive Learning Path

Start your learning journey with personalized guidance through developer roadmaps.

## Purpose
Guide users through structured learning paths based on their role, experience level, and goals.

## How It Works

When you run `/learn`, Claude will:

1. **Assess Your Background**
   - What's your current experience level?
   - What role interests you?
   - What's your timeframe?

2. **Recommend Learning Path**
   - Beginner (0-6 months)
   - Intermediate (6-12 months)
   - Advanced (12+ months)

3. **Provide Structured Modules**
   - Core concepts and fundamentals
   - Practical exercises and projects
   - Hands-on challenges
   - Assessment and feedback

4. **Track Progress**
   - Set milestones and goals
   - Monitor skill development
   - Celebrate achievements

## Example Conversation

```
You: /learn

Claude: Welcome to Developer Roadmap Pro! Let's find your perfect learning path.

1. What role interests you most?
   - Frontend Developer
   - Backend Developer

   - Data & ML Engineer
   - Software Architect

   - Beginner (0-2 years)
   - Intermediate (2-5 years)
   - Advanced (5+ years)

Claude: Perfect! Here's your 6-month backend development roadmap...
```

## Learning Path Modules

### Module 1: Fundamentals (Weeks 1-4)
- Programming language basics
- HTTP and web protocols
- Database fundamentals
- API concepts

### Module 2: Practical Skills (Weeks 5-8)
- Building APIs with Express/FastAPI
- Database design and optimization
- Authentication and security
- Testing and debugging

### Module 3: Advanced Topics (Weeks 9-24)
- Microservices architecture
- Scalability patterns
- DevOps and deployment
- Performance optimization
