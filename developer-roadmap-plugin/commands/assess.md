# /assess - Knowledge Assessment

Evaluate your skills and knowledge across different development domains.

## Purpose
Provide comprehensive assessments to help you understand your current skill level and identify areas for improvement.

## Assessment Types

### 1. **Quick Skill Check** (5-10 minutes)
Short quiz to gauge basic understanding
- 5-10 multiple choice questions
- Immediate feedback
- Topic-specific

### 2. **Domain Assessment** (15-20 minutes)
Comprehensive test of a specific domain
- 15-20 questions mixed format
- Detailed score breakdown
- Recommendations for improvement

### 3. **Full Career Assessment** (45-60 minutes)
Complete skill evaluation across all areas
- All 7 agent domains covered
- Identifies strengths and gaps
- Personalized growth plan

### 4. **Practical Coding Assessment**
Real-world coding challenges
- 2-4 problems
- Various difficulty levels
- Code review and feedback

## Scoring & Feedback

### Score Interpretation
- **90-100%**: Expert level - Consider teaching or mentoring
- **70-89%**: Advanced - Well-rounded knowledge
- **50-69%**: Intermediate - Good foundation, room to grow
- **30-49%**: Beginner - Solid basics, more learning needed
- **0-29%**: Novice - New to the topic, start fundamentals
